# [myd-dev](https://myd-dev.pages.dev)

마이데이터 사업자, 제공자를 위한 API 스펙 탐색 웹 앱

## QuickStart

```bash
git clone https://github.com/myddev/myd-dev
cd myd-dev
npm i
npm run dev # 개발 서버 시작
```

## 기여 방법

코드 스타일은 프로젝트에 정의된 eslint/prettier 설정을 따릅니다.
